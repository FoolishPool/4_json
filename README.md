# Понятный JSON

Скрипт отображает JSON файл, принимаемый как аргумент при запуске, и отображает его 
в понятном виде.

# Быстрый старт


Пример запуска скрипта Linux, Python 3.5:

```
$ python3 pprint_json.py <path to file>
```
